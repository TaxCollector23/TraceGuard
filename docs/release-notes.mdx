## Unreleased — TraceCompress

- **TraceCompress**: prompt compression + output discipline.
  - Modes: Normal, Concise, Bare Mode (strict response rules).
  - Preserved-constraints checklist, removed-redundancy summary, conflict
    detection, response-rules block, output-budget presets (Tiny/Short/Normal/Detailed).
  - CLI: `trg compress-prompt --mode <m> [--output-budget <p>]` and
    `trg run --compress`.
  - Dashboard: **Prompt Compressor** page.
  - Privacy: `[prompt_compression]` config; `prompt_history = false` stores only
    estimates + metadata, never raw prompt text. External-LLM compression is
    opt-in.
  - New endpoints under `/api/prompt-compressor/*`.
- `trg update` self-update command.
- MIT `LICENSE` added.

## v0.1.0 — Local-first MVP

Initial release.

**Core**
- `trg` CLI: `init`, `run`, `dashboard`, `rollback`, `compress-prompt`,
  `daemon start|stop|status`.
- Local Axum daemon bound to `127.0.0.1`, port discovery from `8757`.
- Local SQLite persistence and the local React dashboard (Dashboard, Run
  Timeline, Patch Review, Cost Center, Risk Center, Rollback Center, Utilities).

**Monitoring**
- Run recording with checkpoints, file watching, command output capture.
- Git checkpoints and Git-based rollback.
- Rule-based command guarding (allow / warn / require approval / block).
- Local secret detection with redaction.
- Cost tracking schema with honest partial-data handling.
- Deterministic patch diagnosis.

**Productivity**
- Prompt Compressor (local, deterministic) and output-budget controls.

**Surfaces**
- Public landing site (`apps/landing`, Vercel).
- Mintlify documentation.
- Integration scaffolds: GitHub, Claude Code, Codex, VS Code, Cursor.
- Installers: Homebrew tap, shell script, PowerShell script, optional npm
  wrapper.

> Dates and versions are updated as releases are tagged.
